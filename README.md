# et_install.sh

Script to install et (EternalTerminal) on RHEL7/8 , CentOS7, Fedora , Debian on crostini.

## Environment

- Tested CentOS7 as well, use `et_install_rhel7.sh`.

## Usage

- To install et on RHEL/CentOS7, run 

```bash
./et_install_rhel7.sh 
```
